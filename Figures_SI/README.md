# Figures
Scripts in this directory are used to create the supplementary figures included in the SI.

## Directories
* addFeW - figures including Fe, W Volmer-Tafel pathways
* coverage - plots of adsorbate-adsorbate interactions
* DRC - Degree of Rate Control plots
* Hey_taf_selectivity - selectivty plot of Tafel/Heyrovsky rates
* OHads - plot comparing Volmer barrier with and without OH adsorption on Ni
* electronic_correlations - plots relating energetics and electronic descriptors
